Detailed Description

This is the remote façade pattern class. This class will be in contact with the client side of the application. Managing all of the GET and POST requests, and making use of the ApplicationService pattern.

{@Path} in the class definition is used to define the HTTP path, as well as all of the {@Path} seen bellow in the corresponding methods.
